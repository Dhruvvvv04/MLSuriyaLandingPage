# M.L. Suriya Landing Page

A beautiful, modern landing page for M.L. Suriya featuring two distinct sections:
- **Spiritual Seekers**: For those on a path of spiritual exploration
- **Corporate Professionals**: For those seeking spirituality in business

## Features

✨ **Color Palette**: Premium purple, gold, coral, and rose color scheme from ML Suriya's original design
🎨 **Modern Design**: Glassmorphism effects, smooth animations, and responsive layout
📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
⚡ **Fast**: Built with Vite for optimal performance
🧘 **Spiritual Focus**: Two distinct paths tailored to different audiences

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation

1. Navigate to the project directory:
```bash
cd "/Users/dhruvpatel/Desktop/M.L. Suriya Landing Page"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will open automatically at `http://localhost:5174/`

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `dist/` folder.

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
  --bg: #f7f5fe;
  --p600: #6b35c8;
  /* ... other colors ... */
}
```

### Modifying Content
Edit the HTML sections in `index.html`:
- Update hero content
- Modify card titles and descriptions
- Change contact form fields

### Adding More Sections
Follow the same structure:
1. Add HTML section
2. Create corresponding CSS class
3. Add any interactive JavaScript as needed
